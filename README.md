# Student Result Calculator
This is a simple web-based application developed using HTML, CSS, and JavaScript.
# Features
- Accepts marks for 3 subjects
- Calculates total and average
- Assigns grade based on average
- Simple and clean user interface
# Technologies Used
- HTML
- CSS
- JavaScript
# How to Run
1. Download the project files
2. Open index.html in browser
3. Enter marks and click Calculate
